Ce que nous livrons autour de Metabase

  • Modèle de données : métriques cohérentes, dictionnaire, conventions de noms.
  • Qualité : tests de données, détection d’anomalies, traçabilité.
  • Sécurité : RBAC, groupes, (si besoin) sécurité ligne par ligne.
  • Performance : caches, index, vues matérialisées, refresh maîtrisé.
  • Partage : embeds, exports PDF/CSV, e-mails programmés.
  • Gouvernance : droits, environnements, documentation de bord.
  • Évolutivité : ajout de sources, multi-projets, industrialisation.
  • Style : design moderne, lisibilité, formats FR (€, dates).
