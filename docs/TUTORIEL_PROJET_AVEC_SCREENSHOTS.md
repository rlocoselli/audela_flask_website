# Tutoriel Projet avec screenshots

Ce guide explique le module Projet (Kanban, Gantt, Gouvernance, Reporting, Assistant IA) avec captures ecran.

## 1. Objectif

Montrer un flux complet:
- creation de projet avec IA,
- generation des taches,
- suivi Kanban/Gantt,
- gouvernance,
- reporting,
- partage public,
- verification multilingue.

## 2. Prerequis

- Utilisateur connecte avec acces au module Projet.
- Donnees de test disponibles (ou projet vide).
- Langue active visible dans l interface.

## 4. Parcours pas a pas

### Etape 1 - Ouvrir le module Projet

Action:
- Aller sur `Projet` depuis le menu principal.
- Verifier que les sections `Tableau`, `Responsables`, `Gouvernance`, `Reporting`, `Gantt`, `Notifications` sont visibles.

Capture:

![Project dashboard](tutorial_assets/project/01-project-dashboard.png)

### Etape 2 - Ouvrir l assistant IA

Action:
- Cliquer sur `Creer projet par IA`.
- Verifier que la fenetre IA affiche le chat et le resume de projet.

Capture:

![AI assistant open](tutorial_assets/project/02-ai-assistant-open.png)

### Etape 3 - Remplir le questionnaire guide

Action:
- Choisir `Type de projet`, `Complexite`, `Mode de delivery`, `Contrainte principale`.
- Cliquer `Appliquer au chat`.

Capture:

![AI questionnaire](tutorial_assets/project/03-ai-questionnaire.png)

### Etape 4 - Repondre aux questions intelligentes IA

Action:
- Selectionner des options proposees.
- Tester le bouton `Suggérer d'autres options`.
- Verifier que le texte reste dans la langue active.

Capture:

![AI smart questions](tutorial_assets/project/04-ai-smart-questions.png)

### Etape 5 - Creer le projet et verifier le Kanban

Action:
- Cliquer `Creer projet maintenant`.
- Verifier la creation automatique des cartes et descriptions.

Capture:

![Generated kanban](tutorial_assets/project/05-generated-kanban.png)

![Gantt view](tutorial_assets/project/06-gantt-view.png)

## 5. Checklist de validation finale

- [ ] Aucun melange FR/PT/EN dans les questions IA.
- [ ] Les options IA suivent la langue active.
- [ ] Les descriptions Kanban sont dans la bonne langue.
- [ ] Les taches Gantt sont dans la bonne langue.
- [ ] Les menus module Projet sont traduits.
- [ ] Le partage Kanban fonctionne.
- [ ] Les archives IA se restaurent sans perte.

## 6. Notes de qualite screenshot

- Resolution recommandee: 1600x900 minimum.
- Zoom navigateur: 100%.
- Masquer donnees sensibles avant capture.
- Conserver le meme theme visuel sur toutes les captures.
